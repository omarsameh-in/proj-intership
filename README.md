# InternWay Login Page - React + Bootstrap

صفحة تسجيل الدخول لمنصة InternWay مبنية باستخدام React و Next.js و Bootstrap 5.

## المميزات

### التصميم
- تصميم من لوحتين: لوحة يسرى بتدرج لوني جميل (40%) ولوحة تسجيل الدخول (60%)
- شعار متحرك (IW) مع تأثير floating
- تدرجات لونية جميلة (gradients)
- شريط تمرير مخصص (custom scrollbar)

### الوضع الداكن والفاتح
- تبديل سهل بين الوضع الداكن والفاتح
- حفظ التفضيلات في localStorage
- تصميم متناسق في كلا الوضعين

### دعم متعدد اللغات
- اللغة الإنجليزية والعربية
- دعم كامل لـ RTL (من اليمين لليسار)
- حفظ اللغة المختارة

### متجاوب تماماً
- تصميم متجاوب لجميع الأجهزة
- إخفاء اللوحة اليسرى على الموبايل
- أحجام مناسبة للشاشات الصغيرة

### اختيار الدور
- ثلاثة أدوار: طالب، مرشد، شركة
- أيقونات ملونة مميزة لكل دور
- تغذية بصرية عند الاختيار

### التحقق من النموذج
- التحقق من البريد الإلكتروني
- حقول مطلوبة
- رسائل خطأ واضحة

## الربط مع الخلفية (Backend Integration)

هذا المشروع جاهز للربط مع API. تم إعداد `axios` في `app/lib/api.ts` ليقوم تلقائيًا بإرسال الـ Token المخزن.

### المتغيرات البيئية (Environment Variables)
قم بإنشاء ملف `.env` في المجلد الرئيسي وأضف المتغير التالي:
```bash
NEXT_PUBLIC_API_URL=http://localhost:5000
```
(يمكنك نسخ `.env.example`)

### نقاط الاتصال المطلوبة (Endpoints)

يجب أن يوفر الباك إند الروابط التالية:

1. **تسجيل حساب جديد (POST):**
   - `/auth/register/student`
   - `/auth/register/company`
   - `/auth/register/mentor`
   *(يجب أن تدعم هذه الروابط استقبال البيانات ورفع الملفات في حالة الطالب)*

2. **تسجيل الدخول (POST):**
   - `/auth/login`
   - **البيانات المرسلة:** `{ email, password, role }`
   - **الرد المتوقع:** `{ token: "...", user: { ... } }`
